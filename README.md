# 가스경영 2025 - API 백엔드

Spring MVC와 MSSQL 기반으로 구축된 **가스경영 2025** 프로젝트의 API 백엔드입니다.  
고객 관리, 차량 운행, 충전 관리, 거래 내역 등을 API로 제공합니다.

---

## 🧰 기술 스택

- **Backend Framework**: Spring Boot (Spring MVC)
- **Database**: Microsoft SQL Server (MSSQL)
- **ORM / DB Access**: JdbcTemplate, MyBatis 일부
- **Build Tool**: Maven
- **Language**: Java 11+
